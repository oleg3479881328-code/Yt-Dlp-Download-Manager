# 07 RESULT — Animated Subtitle Video Maker Scope Decision

## Result Date

2026-05-25

## Trigger

Олег уточнил, что проекту нужен не только экспорт файлов субтитров, но и создание собственных роликов с karaoke highlighting (караоке-подсветкой), изменяемыми цветами, анимациями появления, последовательностью текста и положением на экране.

## Researched Direction

Для задачи определено готовое технологическое направление:

- `yt-dlp` — получение доступных исходных субтитров;
- `stable-ts` / `faster-whisper` — распознавание речи и word-level timing (временные метки отдельных слов);
- `Remotion` + `@remotion/captions` — preview/rendering (предпросмотр/отрисовка) анимированных субтитров и финальный экспорт видео.

## Owner Decision

Выбран вариант `A`: добавить `Animated Subtitle Video Maker` (модуль создания роликов с анимированными субтитрами) внутрь существующего личного проекта `Yt-Dlp-Download-Manager`.

## Approved Scope

Новый модуль должен в перспективе позволить:

- использовать собственные ролики пользователя;
- получать или генерировать субтитры;
- подсвечивать произносимые слова в karaoke mode (караоке-режиме);
- варьировать цвет, стиль подсветки, появление текста, последовательность и позицию на экране;
- экспортировать готовый MP4 с burned-in subtitles (вшитыми субтитрами).

## Phase 1 MVP Boundary

Первая реализация ограничивается одним доказуемым сценарием:

1. выбрать собственный ролик;
2. получить или передать временные метки слов;
3. применить один karaoke preset (караоке-пресет);
4. увидеть preview (предпросмотр);
5. экспортировать MP4 с вшитыми субтитрами.

Не входят в первый этап: полный редактор стилей, массовый рендеринг, перевод, определение говорящих и переработка существующего контура загрузок.

## State Separation

- Implementation state (состояние реализации): not started (не начато).
