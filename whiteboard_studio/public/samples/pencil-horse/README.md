# Pencil Horse Approved Visual Package

This package is for Whiteboard Renderer Phase 2.

## Owner-approved source

The owner approved the uploaded pencil horse image as the visual reference.

## Files

- `approved-reference.png` — exact approved reference, resized to 1080x1920.
- `approved-reference-horse-only.png` — reference with the visible hand/pencil area masked out for horse-focused comparison.
- `approved-drawing.svg` — grouped SVG/path trace for draw-on animation.
- `approved-drawing-preview.png` — quick preview of the SVG trace.
- `pencil-horse-scenes.json` — proposed timeline/order for rendering.
- `CODEX_HANDOFF.md` — ready-to-paste task instructions for Codex.

## Recommended use

Codex should use `approved-drawing.svg` and `pencil-horse-scenes.json` as the implementation source.

The PNG reference is not to be rendered as a static pop-in. It is only the approved look.
